## 单调栈

简单理解一个简单的单调栈用法

一个数组, 找到每一个位置左边最近最大的, 右边最近最大的

根据这个特性

```js
const getRecent = arr => {
    
    let stack = [], result = []
    
    for (let i = 0, len = arr.length; i < len; i ++) {
        if (i === 0) stack.push({index: i, value: arr[i]})
        if (stack[stack.length - 1] < arr[i]) { // 即将插入的数字小于栈顶, 就可以收集栈顶的数据了
            let target = stack.pop()
            result.push({
                index: target.index,
                left: stack[stack.length - 1].index,
                right: i
            })
        }
    }
    return result
}
```
